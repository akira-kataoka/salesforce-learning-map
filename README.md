# 🪐 Salesforce ニューロン学習マップ

Salesforce の学習項目を、**神経細胞（ニューロン）のように有機的につながる知識ネットワーク**として可視化した、Three.js 製のインタラクティブ 3D マップです。
関係のあるトピック同士がシナプスでつながり、**光の信号**がその間を流れます。学習の「**深さ**」と「**つながり**」を直感的に掴めることを目指しています。

**🔗 公開サイト:** https://akira-kataoka.github.io/salesforce-learning-map/

## 特徴

- **822 ノード（学習トピック 735 + 種別ハブ 87）** — 製品 (27)・職種 (11)・認定資格 (35)・概念 (14)・学習トピック (735)
- **関連が強いほど近く、弱いほど遠く** — 力学レイアウトで、前提・包含の強い関係は引き寄せ、無関係なものは反発で遠ざける
- **球の大きさ = 学習の重大度** — 生成時に各トピックへ付与した重大度 (sig 1–5) を主軸に、前提被参照数・横断数・内容量を合成し、順位（パーセンタイル）で均等にサイズへ反映
- **学習の深さ（深度）** — 前提チェーンの深さ × レベルから 5 段階（起点／基礎／実践／深化／深奥）を算出。ノードの色に深さの色（水色→紫）を重ね、詳細パネルに深度メーターを表示
- **前提パスの可視化** — ノードを選ぶと「**ここへ至る道**（前提）」を水色、「**ここから拓ける道**（派生）」を緑で照らし、信号がその学びの道を流れる
- **習得トラッキング** — トピックを「習得」でき（ブラウザに保存）、金色に灯る。習得ノード同士の結線も金色に。進捗 HUD「習得の軌跡 X / 735」
- **フロンティア** — 前提をすべて習得した「次に学べる」トピックが緑の縁でやわらかく明滅
- **参考リンク** — 各ノードから Trailhead・公式ヘルプ・Trailblazer コミュニティ・開発者ドキュメント（種別に応じて資格ガイド／製品）へ直接リンク
- **フィルタ / 検索** — 種別・レベルでの絞り込み（種別をオフにすると、その種別だけに繋がるトピックも一緒に消える）、キーワード検索
- ダークモード専用・レスポンシブ・完全自己完結（Three.js r128 をローカル同梱）

## 使い方

1. **ドラッグ**で回転、**ホイール**でズーム、**右ドラッグ**で平行移動
2. **ノードにマウスを重ねる**と概要、**クリック**でそのノードに寄り、右側に詳細・深度・前提／派生の道が開きます
3. トピック詳細の「**習得した — 灯をともす**」で、自分の知識ネットワークを金色に育てられます
4. 凡例で**種別・レベル**を絞り込み、上部の**検索**で項目を探せます

## 構成

```
index.html          エントリ（3D ニューロンマップ）
css/style.css       スタイル
js/sfmap-core.js    データ構築（重大度・深さの算出）・詳細パネル生成
js/app3d.js         Three.js シーン・力学レイアウト・信号・習得・ブルーム・操作
js/vendor/          Three.js r128 一式（ローカル同梱）
data/anchors.js     製品/職種/資格/概念（種別ハブ）のデータ
data/topics.js      学習トピック 735 件（Fable 5 生成・重大度付き）
```
